DelphiWebScript II 

ISAPI DLL

The DelphiWebScript II ISAPI DLL is an ISAPI module that runs on every ISAPI capable web server (tested with PersonalWebServer, IIS 5 and OmniHttpd).

Contents

This DLL processes HTML documents with DWSII scripts embedded in HTML comment tags (<% %>). If a session id is passed in the HTTP request the built-in session manager is activated and preserves the session data (properties of class User) across HTTP requests. If the built-in remote debugger is enabled the DLL connects to the external remote debugger IDE (also part of this package) and processes the script in debugging mode. 

Available script functions:

  • All built-in DWSII utility functions
  • Send(), SendLn(): Send a string to the response.
  • Class "Request" to read the properties of the HTTP-Request that started the script
  • Class "Response" to read and write the properties of the HTTP Response that will be sent if the script was processed.
  • Class "ScriptDoc" to read information about the script file
  • Class "Session" to read and write information about the current user session.
  • Function "TID" returns the session id.  
  • Class "TQuery" provides simple database access to BDE database aliases.

Installation

  • Copy the file dws2.dll to the "scripts" directory of your web server. 
  • If necessary: Configure your web server the way that it passes requested documents with extension .dws to dws2.dll. 

Example: If you use PersonalWebServer (ships with Windows 98) the "scripts" path is "C:\inetinfo\scripts". Copy "index.dws" and the directory "demos" to "C:\inetinfo\wwwroot" and open the link "http://localhost/scripts/dws2.dll/index.dws" in your browser. 

Documentation

Have a look at the example files "index.dws" and the other example files in the "demos" directory.

Configuration

  • Script Tags: <% %>
  • Session Manager: Session cookies disabled, session ID identifier "TID", maximal concurrent sessions 150, session timeout 10 Minutes,
