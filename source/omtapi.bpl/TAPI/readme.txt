PLEASE EXTRACT ALL FILES IN A CLEAN DIRECTORY.
1. RENAME TAPI_XX.INC TO TAPI.INC 
	DEFAULT IS TAPI 22 - ONLY FOR WINDOWS 2000 AND XP !! 
	14 - WINDOWS 95
	20 - WINDOWS NT 4.0 SP3
	21 - WINDOWS 98,NT 4 SP4,AVAILABLE AS A REDISTRIBUTABLE FOR WINDOWS 95. 
	22 - WINDOWS 2000 AND XP
2. CREATE YOUR PACKAGE e.g. TAPI22_D7.bpl [FOR API VERSION 2.2 AND DELPHI 7]
3. ADD ONLY SUBDIR STD AND ACD NOW.
4. COMPILE AND INSTALL
5. ADD THE SUBFOLDER MISC
6. BEFOR YOU MAKE YOUR VERSION CHANGE THE GERMAN WORDS IN THE FORMS.:
	"VERBINDEN ÜBER .." - "CONNECT TO"
	"ANSCHLUß"	    - "LINEDEVICE"	
	"ADRESSE"	    - "ADDRESS ON THE LINE"
	"ANSCHLUßPARAMETER" - "LINEDEVICE CONFIGURATION"
7. INSTALL YOUR PACKAGE
