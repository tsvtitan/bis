Программа предназначена для ввода с клавиатуры мобильного телефона монофонических мелодий. 
Годится для ЛЮБОГО телефона ЛЮБОЙ модели, имеющий синтезатор мелодий звонка.
Программа использует возможность Magic Keys, состоящую в имитации нажатия кнопок телефона.
Для работы программы требуется сценарий нажатий на кнопки телефона и наборы знаков ввода нот для каждой кнопки.
Прилагаются типичные сценарий и наборы знаков для кнопок для условно-нотной записи мелодий для телефона Motorola C350. 
Для других телефонов нужно подготовить сценарий и набор знаков для кнопок - для этого имеются визуальные ссредства.  
Мобильный телефон должен быть подключён к компьютеру.
Программа работает только в паре с программой Easyport версии 3.38.
